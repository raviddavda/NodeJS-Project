Bcards API Documentation

User Endpoints

Endpoint Method Description Permissions
/api/v1/users/login POST Authenticate a user and obtain a session token Any user
/api/v1/users POST Create a new user account Any user
/api/v1/users/:id GET Retrieve details of a specific user Admin or account owner
/api/v1/users GET Retrieve a list of all users Admin only
/api/v1/users/:id PUT Edit the details of a user account Account owner only
/api/v1/users/:id PATCH Change business account status Account owner only
/api/v1/users/:id DELETE Delete a user account Admin or account owner

Business Card Endpoints

Endpoint Method Description Permissions
/api/v1/cards GET Retrieve a list of all business cards Any user
/api/v1/cards POST Create a new business card Business status or Admin account only
/api/v1/cards/my-cards GET Retrieve all cards created by the current user Card owner only
/api/v1/cards/:id GET Retrieve details of a specific card by ID Any user
/api/v1/cards/:id PUT Edit the details of a card Admin or account owner only
/api/v1/cards/:id PATCH Like a card Any user
/api/v1/cards/:id DELETE Delete a card Admin or account owner only
